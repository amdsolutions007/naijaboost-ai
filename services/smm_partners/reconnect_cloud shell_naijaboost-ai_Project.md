# Step-by-Step Guide to Reconnect to Your Project

If your system shuts down or you lose connection, here is the simple, step-by-step guide to get right back to where you were with the naijaboost-ai project:

Step 1: Activate Cloud Shell

Click the "Activate Cloud Shell" button in your Google Cloud console. The terminal and editor will open.

Step 2: Verify Your Files Are Safe

The first command you should always run is ls to list your files and folders. This will immediately show you that the naijaboost-ai folder is still there.

Bash

ls
Step 3: Navigate Back to the Project Directory

Just like we did before, move into your project's directory.

Bash

cd naijaboost-ai
Step 4: Check Your Git Branch Status

To confirm you are exactly where we left off, run this command. It will show you that you are on the feature/backend-scaffold-v1 branch and that your work is up to date.

Bash

git status
Step 5: Re-open Files in the Editor

The editor on the left will likely be empty. Just use the file explorer on the left to click on the naijaboost-ai folder, and then click on any files (main.py, cloudbuild.yaml, etc.) you wish to view or edit again.

Following these steps will always get you back to your work safely and quickly.
